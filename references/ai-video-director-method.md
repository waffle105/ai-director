# AI Video Director Method

## Core Workflow

Use this workflow for short promotional, product, activity, cultural tourism, training, story, or branded videos:

```text
Theme -> Brief -> Creative directions -> Storyboard -> Lens-level prompts -> Asset plan -> Image assets -> Image-to-video prompts -> Hard-shot testing -> Editing plan -> Release versions
```

## Production Principles

- Start with the video purpose, not the tool.
- Build reusable assets before asking a video model to move them.
- Use one shot for one task.
- Keep shots short. For 15 seconds, use 2-4 shots.
- Prefer static or simple camera movement over complex multi-person action.
- Use post-production for Chinese titles, dates, addresses, QR codes, subtitles, and legal claims.
- Make product and character references before motion generation.
- For important videos, generate each shot separately and edit them together.
- Treat a whole-video prompt as a reference only; do not rely on it for final production.

## Creative Direction Pattern

For each theme, create three directions:

1. **Safe version**: easiest to generate and most reliable.
2. **Social-spread version**: stronger visual hook and platform appeal.
3. **Story version**: uses a small narrative to carry the message.

Recommend one direction based on:

- generation feasibility
- message clarity
- platform fit
- asset controllability
- emotional appeal

## Storyboard Rules

Each shot should include:

- time range
- shot size
- visual content
- action
- camera movement
- key message
- generation difficulty
- backup option

Avoid:

- many people interacting in one shot
- complex hand-object operations
- fast walking while holding detailed products
- readable text inside generated video
- logo, QR code, exact dates, or addresses generated by AI
- product transformations that may distort the product

## Hard-Shot Strategy

Identify the hardest shot before generation. Hard shots often include:

- opening boxes
- handing products between people
- detailed food or packaging close-ups
- consistent character movement
- crowded market scenes
- screens with readable content

For each hard shot, provide:

- a simplified version
- a static first-frame alternative
- a camera-movement-only alternative
- a cutaway shot option
- a post-production workaround

## Quality Criteria

A preparation package is complete only when:

- the user can see what the video is for
- each shot has a clear job
- each shot has its own prompt and asset list
- all image assets are named and mapped to shots
- hard shots have backup plans
- post-production text and narration are separated from generated visuals
- release versions are adapted to target platforms
